# 贡献指南

## 如何贡献？

我们欢迎各种贡献形式，包括但不限于以下形式：

- 报告使用过程中遇到的任何错误或非预期行为

- 新的需求和想法

- 缺陷修复

- 测试工作

- 文档撰写

- 新功能或改进

## 代码风格

目前，我们没有标准的代码格式模板。

在提交 Pull Request 之前，请确保格式化待提交的代码：

- 使用 IDE 格式化 Java 代码

- 使用 `npm run format` 格式化前端代码
