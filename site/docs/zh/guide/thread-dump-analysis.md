# 线程快照分析

## 介绍

线程快照分析是一种用于分析线程相关问题的技术手段。线程快照可以提供进程某一时刻所有线程的状态和调用栈等信息。该功能可以帮助开发者分析线程泄漏、死锁、CPU
高等问题。
